Privacy Policy for ArchCraft GPT

Last updated: —

Overview

ArchCraft GPT helps you look up Arch Linux packages (official repos and AUR) and Linux kernel releases. To answer your questions, the GPT may query public, unauthenticated endpoints listed below. No personal data is collected, stored, or sold by the creator of this GPT.

Public APIs Used

  • archlinux.org
  • aur.archlinux.org
  • kernel.org
  • pkgbuild mirror

Endpoints

archlinux.org

  • GET /packages/search/json/?name=<term>
  • GET /packages/{repo}/{arch}/{pkgname}/json/

aur.archlinux.org (RPC v5)

  • GET /rpc/?v=5&type=search&arg=<term>
  • GET /rpc/?v=5&type=info&arg[]=<pkgname>
  • GET /rpc/?v=5&type=multiinfo&arg[]=<pkg1>&arg[]=<pkg2>…

www.kernel.org

  • GET /releases.json

geo.mirror.pkgbuild.com

  • Static content/file listings.

What Data Is Sent

Only the query parameters needed for your request:

  • Package names (e.g., hyprland)
  • Repo/arch when relevant (e.g., extra/x86_64)

No PII is sent. No keys or auth required.

Data Storage & Retention

  • No server-side databases are maintained.
  • API requests are transient only.
  • No personal data retained.

Cookies & Tracking

This page sets no cookies. Linked sites may use their own policies.

Security

  • Only HTTPS endpoints are used.
  • No secrets or tokens involved.
  • All data exchanged is non-PII.

Third-Party Links

Linked resources (Arch, AUR, kernel.org) have independent policies. The creator isn’t responsible for them.

Jurisdiction & Legal Basis

This is an informational, non-commercial project. Queries are processed under legitimate interest with public data only. No profiling or marketing.
